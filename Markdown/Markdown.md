# Markdown記法

拡張子：.md<br>
半角空白2つで改行
<br>

## 見出し（タイトル）
<details>
<summary>詳細</summary>

```
# レベル1見出し
## レベル2見出し
### レベル3見出し
#### レベル4見出し
##### レベル5見出し
###### レベル6見出し
```
</details>
<br>

## 太字・斜体・訂正線
<details>

```
**太字**
*斜体*
~~訂正線~~
```
【表示例】<br>
**太字**<br>
*斜体*<br>
~~訂正線~~<br>
</details>
<br>

## テキストカラー
<details>

```
<span style = "color : red;">赤文字</span>
```
【表示例】<br>
<span style = "color : red;">赤文字</span>
</details>
<br>

## コメントアウト
<details>

```
<!-- テキスト -->
```
</details>
<br>

## 箇条書きリスト
<details>

```
- リスト
```
【表示例】<br>
- リスト1
- リスト2
  - サブリスト2.1
</details>
<br>

## 番号付きリスト
<details>

```
1. 番号リスト
```
【表示例】<br>
1. 番号リスト1
2. 番号リスト2
</details>
<br>

## テーブル
<details>

```html
<table border>
  <tr>
    <th>th</th>
    <th>th</th>
  </tr>
  <tr>
    <td>td</td>
    <td>td</td>
  </tr>
</table>
```
【表示例】<br>
<table border>
  <tr>
    <th>name</th>
    <th>age</th>
    <th>address</th>
  </tr>
  <tr>
    <td>田中</td>
    <td>20</td>
    <td>東京</td>
  </tr>
  <tr>
    <td>斎藤</td>
    <td>25</td>
    <td>大阪</td>
  </tr>
</table>

[テーブルの詳細について](https://www.sejuku.net/blog/49377)
</details>
<br>
